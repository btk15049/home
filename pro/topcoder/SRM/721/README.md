## Hard
フローを貼るだけ．  
とはいってもサンプルに少し罠があり，距離t以下の頂点同士に辺を貼ってマッチング という解法だと``{0,0,0,0},{0,1,2},2``みたいなケースで落ちる．  
各ターン各頂点に存在できるトークンは高々一つという条件を丁寧に実装して流すと簡単に解ける．
