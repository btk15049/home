10進数の桁和でmod3やmod9が求められることの一般化で，k進数の桁和でmod k-1が求められる  
また，10^12<=lcm(2,3,4,...,30)>=10^12 が成り立つので，中国剰余定理を使うと，10^12以下のmod条件を満たす整数が唯一つ定まる．あとは桁和がそうなるか確認すればOK
